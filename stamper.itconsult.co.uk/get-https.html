Stamper Timestamping Service – Get Signatures via HTTPS

Signatures may be retrived using HTTPS, the signatures being available immediately after having been generated, and permanently thereafter. This is achieved by specifying the stamper id of the signature. The details below include real signatures. Change the parameter to the HTTPS get for the signature number wanted. In all cases, a 404 error will be returned if the stamper id does not exist.

https://timestamp.stamper.itconsult.co.uk/get-sig-info?stamperid=2114411 will return details of the available signature types for the specified stamper id. They are returned as hyperlinks, so that they would work in a browser. For example:- 

$ curl https://timestamp.stamper.itconsult.co.uk/get-sig-info?stamperid=2114411
<A HREF="https://timestamp.stamper.itconsult.co.uk/get-sig-pgp?stamperid=2114411">pgp signature 2114411.asc</A><BR>
<A HREF="https://timestamp.stamper.itconsult.co.uk/get-sig-tsa?stamperid=2114411">tsa signature 2114411.tsr</A><BR>


https://timestamp.stampertest.itconsult.co.uk/get-sig-pgp?stamperid=2114411 will return the ascii armored signature for the specified stamper id. For example:- 

$ curl https://timestamp.stamper.itconsult.co.uk/get-sig-pgp?stamperid=2114411

https://timestamp.stampertest.itconsult.co.uk/get-sig-tsa?stamperid=2114411 will return the rfc3161 TSR repsponse for the specified stamper id, which is binary and is best saved to a file. For example:-
