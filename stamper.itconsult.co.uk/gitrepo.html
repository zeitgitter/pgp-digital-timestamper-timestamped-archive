Stamper Timestamping Service – Git Repository

All of Stamper's signatures, including daily and weekly feedback, are available in a GIT repository at:-

        https://git.stamper.itconsult.co.uk/

which is updated daily with a commit signed by Stamper's management key. It is available using a Git client and does not offer a browser interface.

The directory structure comprises:-

  • /stamper-files/ - containing files by year showing the highest stamper id on each date
  • /stamper-files/daily - containing daily feedback signatures, one per day
  • /stamper-files/weekly - containing weekly feedback signatures, one per week
  • /stamper-files/signatures - containing all signatures, split across subdirectories to allow for ten thousand signatures per directory
